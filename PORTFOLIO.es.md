# Framework de Automatización QA - Portafolio

Versión en inglés: [PORTFOLIO.md](PORTFOLIO.md)

Fuente de verdad: inglés. Esta versión en español existe para presentación local, entrevistas y comunicación bilingüe.

## Resumen del proyecto

Este es un framework de automatización QA listo para producción, extraído de entornos empresariales. Demuestra arquitectura de pruebas de nivel profesional para servicios financieros, salud y otros sectores regulados.

## Qué hace que este proyecto destaque

### 🏆 Arquitectura empresarial
- Escalable a más de 100 casos de prueba
- Ejecución paralela para retroalimentación rápida
- Reportería y métricas completas
- Consideraciones de seguridad y cumplimiento
- Probado en entornos regulados

### 🎯 Mejores prácticas implementadas
- **Page Object Model**: menos fragilidad y mejor mantenibilidad
- **BDD con Cucumber**: escenarios legibles por negocio
- **Ejecución paralela**: concurrencia de 3 hilos por defecto
- **Métricas de calidad**: cobertura con JaCoCo
- **Escaneo de seguridad**: detección de CVE en dependencias
- **Logging completo**: trazabilidad de toda la ejecución

### 💡 Patrones reales
- Gestión de configuración por ambiente
- Estrategias de manejo de datos de prueba
- Integración con Allure para stakeholders
- Seguimiento de línea base de rendimiento
- Estándares de colaboración en equipo

## Excelencia técnica

### Organización del código
```
Page Objects          → Encapsulación y mantenibilidad
Step Definitions      → Separación de lógica de negocio
Utilities             → Infraestructura reutilizable de pruebas
Configuration         → Configuración específica por ambiente
```

### Métricas de calidad
- **Cobertura**: monitoreada con JaCoCo
- **Rendimiento de pruebas**: línea base por escenario
- **Paralelismo**: 3 hilos concurrentes (configurable)
- **Reportes**: Allure con trazabilidad completa
